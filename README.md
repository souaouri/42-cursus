# 42 Cursus Projects

This repository contains my solutions and implementations for various projects from the 42 School curriculum.

## Structure

### Core Curriculum Projects
- **[fract-ol/](fract-ol/)** - Fractal visualization program using MiniLibX
- **[ft_printf/](ft_printf/)** - Custom implementation of the printf function
- **[get_next_line/](get_next_line/)** - Function to read lines from file descriptors
- **[libft/](libft/)** - Custom C library with essential functions
- **[philosophers/](philosophers/)** - Threading and synchronization project
- **[pipex/](pipex/)** - Shell pipe mechanism recreation
- **[push_swap/](push_swap/)** - Sorting algorithm optimization project

### C++ Modules
- **[cpp_modules/](cpp_modules/)** - C++ learning modules (CPP 00-09)

## Getting Started

Each project directory contains its own Makefile and can be compiled independently. Navigate to the specific project directory and run:

```bash
make
```
